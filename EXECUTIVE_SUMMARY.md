## Methodology

SQL-based threat hunting against authentication logs and asset inventory, with
detections tagged to MITRE ATT&CK:

- **T1078 / T1078.002 (Valid Accounts / Domain Accounts)** — terminated-account
  reuse, unfamiliar country, impossible travel. *(The terminated-account case is
  T1078.002 domain accounts, corrected from an earlier T1078.004 cloud-accounts
  label — this dataset is on-prem domain authentication.)*
- **T1110.001 (Brute Force)** — failures-then-success and failed single-account
  brute force.
- **T1110.003 (Password Spray)** — one source against many accounts.

Severity maps onto ADTE's 0–100 verdict bands; this lab is the manual precursor
to that engine (see the README severity-model section). All queries reproducible
via `sql/detection_queries.sql`.

**Confidence:** measured — see [docs/METRICS.md](docs/METRICS.md).
**Limitations:** no network flow data (exfiltration scope unknown); country-level
geolocation only; synthetic low-noise dataset (production precision will be lower).
